# Terraform with AWS

## Install

```bash
cd ~/Downloads
wget https://releases.hashicorp.com/terraform/1.6.2/terraform_1.6.2_linux_386.zip
unzip terraform_1.6.2_linux_386.zip
sudo mv terraform /usr/local/bin/
sudo rm terraform_1.6.2_linux_386.zip
# verify
terraform --version
# Create alias
alias tf=terraform
```
